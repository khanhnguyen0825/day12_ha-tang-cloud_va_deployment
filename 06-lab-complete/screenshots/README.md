# Screenshots Needed

Place the required submission screenshots in this folder:

- `dashboard.png`: deployment platform dashboard
- `running.png`: service status (running/healthy)
- `test.png`: API test results (health/auth/rate-limit)

Tips:
- Make sure the URL and status are visible in screenshots.
- Avoid exposing secrets (API keys, tokens).
- Keep each file under ~2 MB for easier upload.
